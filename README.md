# 台北一日遊網站

## 網站功能
* 查看、搜尋台北景點
* 個別景點介紹頁面
* 加入會員 可線上預定導覽行程
* 串接Tappay金流付款，線上付款完成行程預定
* 歷史訂單查看


## 使用技術
- 使用AWS EC2雲端架站、資料庫使用MySQL
- 前端 HTML, CSS, JS
- 後端 Python - Flask Framework
- 部署nginx反向代理、Docker 
- Tappay金流串接
- SSL憑證實踐HTTPS
- RESTful API 設計架構
- MVC 設計模式
